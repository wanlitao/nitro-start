# Nitro starter

Look at the [nitro quick start](https://nitro.unjs.io/guide#quick-start) to learn more how to get started.

## Env Settings
### For Development
add .env file in root directory
```bash
PORT=3080
HOST="localhost"
```

### For Production
add .env.production file in root directory
```bash
NITRO_PORT=3280
NITRO_HOST="127.0.0.1"
```
